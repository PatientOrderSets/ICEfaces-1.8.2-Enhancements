panelConfirmation

The confirmation panel is a component that pops up a modal confirmation dialog. It allows the command button and command link components to ask for confirmation before proceeding with the selected action. If the 'Accept' button is selected, action will proceed; if the 'Cancel' button is selected, no action will take place.

You can specify, among other things, texts for the header, message and buttons.

Example

The following exampe show how a command button can invoke a confirmation dialog:


    <ice:commandButton panelConfirmation="deleteConfirmPanel" value="#{confirmation.deleteLabel}" actionListener="#{confirmation.delete}"/>
    ......
    <ice:panelConfirmation id="deleteConfirmPanel" title="#{confirmation.deleteLabel}" message="Do you really want to delete?" acceptLabel="Yes" cancelLabel="No"/>
