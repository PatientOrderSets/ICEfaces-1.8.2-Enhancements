rowSelector

The Row Selector tag enables row selection for DataTables. The Row Selector has the following attributes

  • value The boolean value that indicate this row is selected. This value must be defined in the 'var' of the dataTable. Required
  • multiple When true more then one row can be selected. Default is false
  • enhancedMultiple When true offers following selection model:
    • "Click" selects clicked row and deselects rest
    • "Click + Ctrl" flip the selection of clicked row and keeps previous selection intact
    • "Click + Shift" selects all rows from last clicked row to newly clicked row, and deselect rest
    • "Click + Shift + Ctrl" selects all rows from last clicked row to newly clicked row and keeps previous selection intact
  • selectionListener The listener method to call when a row is clicked
  • selectedClass The CSS style to apply when a row is selected
  • mouseOverClass The CSS style to apply when the mouse is over a row.
  • toggleOnClick Disable row selection on mouse click.
Example

The following code shows how to create a rowSelector component with either single or multiple selection, custom styling, and a selectionListener that is invoked in the INVOKE_APPLICATION phase:

    <ice:dataTable var="person" value="#{scrollerList.persons}"
                   rows="15" columnClasses="tableCol">
        <ice:column rendered="#{table.renderFirstName}">
            <ice:rowSelector value="#{person.selected}" multiple="#{table.multipleSelection}"
                             selectedClass="tableRowSelected" mouseOverClass="tableRowMouseOver"
                             immediate="false" selectionListener="#{table.rowSelection}"/>
            <f:facet name="header">
                <ice:outputText value="#{msgs['component.table.column1']}"/>
            </f:facet>
            <ice:outputText value="#{person.firstName}"/>
        </ice:column>
    </ice:dataTable>
